# log-analyzer
Log Analyzer é uma ferramenta em Python para análise de logs de servidores (SSH, Apache, Nginx), com foco em segurança cibernética. Detecta padrões suspeitos como brute force e acessos inválidos, exportando relatórios em JSON, HTML e gráficos. Inclui testes automatizados, CI/CD e Docker para portabilidade.
